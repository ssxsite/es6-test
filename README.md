## 项目介绍
此项目为平时练习ES6所用项目
1.str.js 模板字符串

2.function.js  箭头函数

3.object.js 操作对象的一些新方法

4.destruct.js 解构

5.class.js es6 class 是es5 构造函数和构造函数的prototype的语法糖，目的是为了让面向对象编程更容易理解。

6.module.js es6 新增的模块的语法糖，用import,export，告别了之前的AMD,CMD的不统一的写法。
